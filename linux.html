📌 常见问题的解决方案

  • 🔸 NOSPLIT / File has unexpected size → 网络缓存代理干扰，换清华源 + 降级 HTTP 解决
  • 🔸 apt install 进度条卡死 → HTTPS 被拦截，临时切 NAT 模式安装，装完改回桥接
  • 🔸 FileZilla 连接被拒绝 → 未装 openssh-server，安装后解决
  • 🔸 FileZilla 身份验证失败 → 确认用户名为 vboxuser，重置密码后解决
  • 🔸 同一局域网其他设备无法访问 → 确认 Apache 监听 0.0.0.0:80，UFW 关闭，判断为路由器 AP 隔离所致（不影响实验评分）
