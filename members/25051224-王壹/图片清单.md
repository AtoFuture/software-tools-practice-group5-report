# 实验截图文件夹说明

把本压缩包解压到你的 Hexo 项目根目录，例如：

C:\Users\Public\software-tools-practice\web-report\

解压后目录结构应为：

web-report/
└─ 实验截图/
   ├─ 01-git-install-version.png
   ├─ 02-node-npm-version.png
   └─ ...
