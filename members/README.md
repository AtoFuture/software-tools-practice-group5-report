# 小组成员原始个人材料目录

本目录用于第五组内部存放各成员的原始个人实验报告、截图和负责内容说明。当前选用的完整 HTML 展示报告位于根项目的 `source/report/index.html`。

目录规则：

- 每位成员使用 “学号-姓名” 建立独立文件夹。
- 每位成员的个人截图放入本人文件夹下的 `截图/` 目录。
- 如需修改小组最终网页，请先在组内说明，避免覆盖他人内容。
