# 软件开发工具实践短学期项目

这是“软件开发工具实践”课程的实验报告静态网站源码。项目使用 Hexo 编写和生成实验报告网页，后续可部署到 GitHub Pages 或 Linux + Nginx。

## 项目目标

- 完成 8 个实验的过程记录、结果截图和知识总结。
- 使用 Git 管理源码和报告修改历史。
- 将实验报告生成为可访问的静态网页。
- 准备最终答辩所需的源码包、网页成果和 PPT。

## 本地运行

```bash
npm install
npm run server
```

默认访问地址：

```text
http://localhost:4000
```

## 构建静态文件

```bash
npm run build
```

生成结果在 `public/` 目录。

## 实验目录

1. 网络软件基础
2. Web 软件开发环境
3. 代码编辑器
4. Web 站点构建
5. 虚拟机安装与使用
6. Linux 环境配置
7. 远程登录管理
8. 软件部署
