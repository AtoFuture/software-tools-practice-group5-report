本项目用于完成《软件开发工具实践》短学期考核。网站内容围绕 Web 开发环境、代码编辑与版本管理、虚拟机、Linux 配置、远程管理和 Web 部署展开。

## 交付目标

- 用 Hexo 生成静态实验报告网站。
- 用 Git 记录项目开发过程。
- 将网站部署到 GitHub Pages 或 Linux + Nginx 环境。
- 在答辩中展示实验过程、关键配置、问题解决和最终成果。

## 报告写作结构

每个实验页面按照下面结构补充：

- 实验目的
- 实验环境
- 实验过程
- 实验结果
- 知识总结
- 出现问题与解决方案
- 心得体会
