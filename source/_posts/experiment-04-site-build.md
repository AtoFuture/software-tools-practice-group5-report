## 实验目的

使用 Hexo 构建静态实验报告网站，理解静态站点生成器的基本工作流。

## 实验环境

- Hexo 项目：`software-tools-practice`
- 源码目录：`source/_posts`
- 构建输出目录：`public`

## 实验过程

新建文章：

```bash
hexo new "实验四 Web 站点构建"
```

本地预览：

```bash
hexo server
```

生成静态文件：

```bash
hexo clean
hexo generate
```

## 知识总结

- Hexo 将 Markdown 文件渲染成 HTML。
- `source/_posts` 保存文章源文件。
- `public/` 是最终可部署的静态网页。
