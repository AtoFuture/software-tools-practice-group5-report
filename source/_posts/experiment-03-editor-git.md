## 实验目的

掌握 VS Code 的基本使用、Markdown 文档编写与 Git 版本管理流程。

## 实验环境

- 编辑器：Visual Studio Code
- 插件：Markdown All in One、Markdown Preview Enhanced、Paste Image
- 版本管理：Git

## 实验过程

配置 Git 用户信息：

```bash
git config --global user.name "Steven070502"
git config --global user.email "your-email@example.com"
```

常用 Git 流程：

```bash
git init
git status
git add .
git commit -m "初始化实验报告网站"
```
