## 实验目的

掌握 VirtualBox 虚拟机安装 Ubuntu Server 的流程，理解 NAT、桥接和 Host-Only 网络模式。

## 实验环境

- 虚拟机软件：VirtualBox
- 客户机系统：Ubuntu Server 22.04 LTS
- 网络模式：NAT + Host-Only

## 实验过程

1. 下载 Ubuntu Server 镜像。
2. 在 VirtualBox 中创建虚拟机。
3. 分配内存、CPU 和虚拟硬盘。
4. 配置 NAT 和 Host-Only 双网卡。
5. 安装 Ubuntu Server。
6. 克隆虚拟机用于多机实验。

## 知识总结

- NAT 适合虚拟机访问外网。
- Host-Only 适合宿主机和虚拟机互通。
- 克隆虚拟机可以快速搭建多节点实验环境。
