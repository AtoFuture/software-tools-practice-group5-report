## 实验过程

检查工具版本：

```bash
node -v
npm -v
git --version
```

安装 Hexo CLI：

```bash
npm install -g hexo-cli
```

初始化项目：

```bash
hexo init software-tools-practice
cd software-tools-practice
npm install
hexo server
```
